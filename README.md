# LMS
Least mean squares filtering
Julia code for simple adaptive filtering problem. 
Using <a href="https://en.wikipedia.org/wiki/Least_mean_squares_filter">LMS</a> filter

<h3>Running</h3>

``
julia test1.jl
``
